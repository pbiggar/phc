README

This package contains everything necessary to run javascript code as well as further enhance the javascript parser/runtime.
If you only intend to run javascript code, the following files are not needed and can safely be removed:
- js.l
- js.y
- parse/metascanner.y
- parser/generator.so.php

This package contains a few examples. You can delete those if you choose as well:
- example1.php
- example2.php
